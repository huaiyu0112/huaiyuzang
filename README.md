
# Personal Profile Website

This repository hosts my personal academic and professional profile website, built as a single-page HTML application and deployed using GitHub Pages.

## About

This is a static HTML profile page built with plain HTML/CSS — no frameworks or dependencies required. It includes:

- **About** — bio and research interests
- **Selected Publications** — with DOI links
- **Work Experience**
- **Software** — R code for Bayesian causal inference ([BNP-causal](https://github.com/huaiyu0112/BNP-causal))
- **Selected Presentations**
- **Education**, **Skills**, and **Honors & Service**

## Usage

The site is a single `index.html` file. To run locally, simply open it in a browser:

```bash
open index.html
```

## Deployment

Hosted via [GitHub Pages](https://pages.github.com/). Any push to the `main` branch automatically updates the live site.
